🎵 Exportify Playlist Merger

📌 Before you start:
  • Download the same playlist from both Exportify services:
  • 🌐 exportify.app – this will be your first file (File 1).
  • 🌐 exportify.net – this will be your second file (File 2).
  • Make sure you export the identical playlist (same tracks) from both sites.
  • Do not rename the files; use them as downloaded.
⚠️ Important: The merge is performed on Track URI. If a track appears in only one file, it will still appear in the output with empty cells for the missing file's columns. This ensures you don't lose any data.
Made for Exportify users · All processing happens in your browser – files never leave your device.
